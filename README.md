# InvoiceKit 🧾

A free, no-signup GST invoice generator for Indian freelancers and small businesses.

## Features
- GST support (IGST / CGST+SGST)
- Live preview as you type
- One-click PDF via print
- Fully offline — no server, no database
- Mobile friendly

## How to Use
Just open index.html in any browser — no installation needed.
